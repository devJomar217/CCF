What is HTML?

In summary, HTML is a language used to structure the content of web pages and create a hierarchical relationship between different elements on the page. By using HTML, web developers can define headings, paragraphs, images, links, forms, and other elements that collectively make up a web page. When a user opens an HTML document in a web browser, the browser interprets the HTML code and renders the content visually, allowing users to interact with the web page and navigate through the hyperlinks provided.

What is HTML?

  • HyperText: "HyperText" refers to text that contains links, allowing users to navigate between different pieces of content. In the context of the web, hyperlinks (or simply links) are used to connect web pages and resources, enabling users to jump from one page to another by clicking on these links.
  • Markup: "Markup" refers to the use of special tags and codes to define the structure and presentation of a document. In HTML, the document is marked up using various elements represented by tags, which define how different parts of the content should be displayed or interpreted.
  • Language: "Language" refers to the fact that HTML is a set of rules and conventions that both web browsers and developers understand. It is a standardized way of describing the content and layout of a web page, making it possible to create consistent and well-structured web pages across different platforms and devices.
  • HTML consists of a series of elements

A Simple HTML Document

Sample

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

Sample Explained

  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document
  • The <html> element is the root element of an HTML page
  • The <head> element contains meta information about the HTML page
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

What is an HTML Element?

An HTML element is defined by a start tag, some content, and an end tag:

<tagname> Content goes here... </tagname>

The HTML element is everything from the start tag to the end tag:

<h1>My First Heading</h1>
<p>My first paragraph.</p>
Start tag Element content End tag
<h1> My First Heading </h1>
<p> My first paragraph. </p>
<br> none none

Note: Some HTML elements have no content (like the <br> element). These elements are called empty elements. Empty elements do not have an end tag!

Web Browsers

The purpose of a web browser (Chrome, Edge, Firefox, Safari) is to read HTML documents and display them correctly.

A browser does not display the HTML tags, but uses them to determine how to display the document:

HTML Page Structure

Below is a visualization of an HTML page structure:

<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>

Note: The content inside the <body> section will be displayed in a browser. The content inside the <title> element will be shown in the browser's title bar or in the page's tab.
